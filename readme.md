[![Build Status](https://github.com/RisenCodeLab/meToChrist/actions/workflows/ci.yml/badge.svg?branch=dev)](https://github.com/RisenCodeLab/meToChrist/actions/workflows/ci.yml)

Source code for [meToChrist.com](https://meToChrist.com)

## to build and run locally

Install [bun 1.3][bun] and [yarn classic 1.22][yarn] then:

    $ git clone --branch=dev https://github.com/RisenCodeLab/meToChrist.git
    $ . meToChrist/bootstrap
    $ yarn run start

You should now be able to point your browser at `localhost:7777`

[bun]: https://bun.com
[yarn]: https://classic.yarnpkg.com
